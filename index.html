Real-time Traffic Sign Board Detection and Recognition

The project aims at using OpenCV for detecting region, later the ROI thus identified is fed to the trained model, for recognizing the appropriate class of the traffic sign. The model consumes videos as input and masks the identified results on to the input video.

Road extraction from high resolution satellite images

The main objective of this project was to develop a system that could perform pixel-wise classification of images, to extract an image containing only the road network. The system was designed and tested for the Massachusetts Road data set. It used the U-Net model to identify and build a separate image of just the network of roads.

Integration of Google Assistant to INSZoom web application

The project was intended to integrate Google assistant to an INSZoom application so as to service the Lawyers, Paralegals, and other stake holders with the facility of navigating through the web app, filling the forms and retrieval of desired information all by voice command.

Automated Thermocol Cutting Machine

The purpose of this project was to develop, a stand-alone system that took input of shape from the user through an Android application, which again dumped the corresponding code from the Amazon’s AWS Server on-to the Arduino micro-controller used on the thermocol cutting machine, this controlled the actuator of the system developed so as to cut the selected shape out on the thermocol sheet.
